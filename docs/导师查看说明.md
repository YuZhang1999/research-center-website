# 课题组网站查看说明

## 项目概述
本项目为"云南省工业废水光催化处理工程技术研究中心"课题组网站，已完成基本框架建设和页面模板开发。网站采用React.js + Ant Design技术栈，包含完整的响应式设计和多个功能模块。

## 快速查看方式（推荐）

### 方式一：使用Python快速查看（最简单）
如果您的电脑已安装Python（大多数电脑已预装），请按以下步骤操作：

1. **打开项目文件夹**
   - 解压下载的项目文件包
   - 进入解压后的文件夹

2. **启动网站服务器**
   - 双击运行 `启动网站.bat` 文件
   - 或者打开命令行，执行：
     ```bash
     python -m http.server 8080
     ```

3. **访问网站**
   - 打开浏览器
   - 访问地址：http://localhost:8080/frontend/build
   - 或直接访问：http://localhost:8080

### 方式二：使用Node.js查看
如果您的电脑已安装Node.js，请按以下步骤操作：

1. **安装依赖（仅需一次）**
   ```bash
   cd frontend
   npm install
   ```

2. **构建网站（如果未构建）**
   ```bash
   npm run build
   ```

3. **启动开发服务器**
   ```bash
   npm start
   ```

4. **访问网站**
   - 打开浏览器访问：http://localhost:3000

## 常见问题解答

### Q1: 打开网站后页面显示空白？
- 检查是否已启动服务器（命令行显示"Server started"）
- 检查浏览器地址是否为 http://localhost:8080/frontend/build
- 尝试清除浏览器缓存后刷新页面

### Q2: 启动脚本无法运行？
- 确保Python已安装（打开命令行输入 `python --version` 检查）
- 如果提示"python不是内部命令"，请尝试使用 `python3` 或安装Python
- 或者使用Node.js方式查看

### Q3: 如何查看网站源代码？
- 网站源代码位于 `frontend/src/` 目录
- 主要页面在 `frontend/src/pages/` 目录
- 组件在 `frontend/src/components/` 目录

### Q4: 网站内容可以修改吗？
- 可以，所有内容都是示例数据，可轻松替换
- 图片资源需要替换为实际图片
- 文本内容可直接在代码中修改

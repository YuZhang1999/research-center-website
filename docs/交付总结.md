## 导师查看指南

### 最简单的方式（推荐）
1. **解压项目文件包**
2. **双击运行 `启动网站.bat`**
3. **打开浏览器访问：http://localhost:8080/frontend/build/**

### 备选方式
- **使用Python**：`python -m http.server 8080`
- **使用Node.js**：进入 `frontend` 目录，运行 `npm install && npm start`

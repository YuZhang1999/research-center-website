- **样式方案**: CSS3 + Ant Design 样式系统

## 快速启动

### 前端启动步骤
1. 进入前端目录：
   ```bash
   cd frontend
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm start
   ```

4. 访问网站：
   - 打开浏览器访问：http://localhost:3000

### 后端启动步骤（待开发）
1. 进入后端目录：
   ```bash
   cd backend
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动服务器：
   ```bash
   npm run dev
   ```
